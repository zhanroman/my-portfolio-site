Контекст и исходные данные

Заявка пришла с биржи: нужен англоязычный сайт Reputro для управления репутационными кампаниями на внешних площадках. Клиент описал стилистику через несколько ключевых слов (tech, SaaS, clean), без макетов и детальной визуальной спецификации. Задача: за короткое окно времени получить полностью рабочий продукт на собственном WordPress-темплейте.

  • Срок: окно 3 дня от старта до продакшна.
  • Ожидание: проект примут без итераций.
  • Язык: весь интерфейс на английском.
  • Функционал: калькулятор кампаний, форма заказа, блог, policy страницы, подготовка к платежам.

Что вошло в поставку

  • Кастомная тема без page builder.
  • Tailwind (production purge).
  • ACF для тарифов, FAQ, текстов.
  • Динамический pricing tiers.
  • AJAX форма заказа + Stripe redirect.
  • Success / Failed order pages.
  • Блог + база для policies.

Подход к реализации

Решение опирается на принцип «минимальное достаточное ядро»: кастомная тема вместо громоздких конструкторов, utility-first стили, предсказуемая файловая структура и чёткое разграничение клиентской логики и серверной обработки.

  • Tailwind для быстрого UI без избыточных CSS слоёв.
  • ACF: параметры тарифов и FAQ редактируются без правки кода.
  • Динамическая форма с localStorage восстановлением.
  • Трёхуровневый pricing по количеству запросов.
  • Подготовлена инфраструктура для webhook Stripe (следующая стадия).
  • Роутинг success/fail с передачей ID заказа.
Принцип: максимум результата за единицу времени без компромисса в качестве – достигается за счёт упрощённого технологического стека и отказа от лишних абстракций.

Высокоуровневая архитектура

  3. AJAX → серверная валидация.
  4. Создание внутренней записи заказа.

  7. Email и сохранённые метаданные.

Почему Tailwind

Utility-классы сокращают когнитивные переключения между CSS и HTML, purge снижает вес стилей, а отсутствие сторонних визуальных фреймворков упрощает поддерживаемость.

Форма и интеграция Stripe

Форма построена модульно: визуальный слой отделён от AJAX-обработчика, что упрощает миграцию на headless при необходимости.

  • Nonce-проверка и фильтрация входящих данных.
  • Динамическая генерация полей для пользовательского контента.
  • Отключение кнопки во время обработки.
  • Унифицированный вывод сообщений статуса.
  • Готовность к добавлению webhook подтверждения.
Стратегия оплаты: быстрый redirect на Stripe + последующая эволюция в полноценный серверный пост-процессинг с верификацией сигнатур.

Tier pricing (JS)

function getUnitPrice(q){if(q>=50)return 7.99;if(q>=10)return 8.99;return 9.99;}
function recalc(){
  const q=+quantityInput.value||1;
  const p=getUnitPrice(q);
  summaryQuantity.textContent=q;
  summaryPricePerReview.textContent='$'+p.toFixed(2);
  summaryTotalPrice.textContent='$'+(q*p).toFixed(2);
}

AJAX обработчик (PHP)

add_action('wp_ajax_reputro_submit_order','reputro_submit_order');
add_action('wp_ajax_nopriv_reputro_submit_order','reputro_submit_order');
function reputro_submit_order(){
  if(!isset($_POST['nonce'])||!wp_verify_nonce($_POST['nonce'],'reputro_order_nonce')){
    wp_send_json_error('Security check failed');
  }
  $quantity=intval($_POST['quantity']??0);
  if($quantity<1||$quantity>1000){wp_send_json_error('Quantity out of range');}
  $url=esc_url_raw($_POST['url']??'');
  if(!filter_var($url,FILTER_VALIDATE_URL)){wp_send_json_error('Invalid URL');}
  $price=reputro_calculate_price($quantity);
  $order_id=wp_insert_post([
    'post_type'=>'reputro_order',
    'post_status'=>'draft',
    'post_title'=>'Order '.time(),
    'meta_input'=>[
      'quantity'=>$quantity,
      'price'=>$price,
      'profile'=>$url
    ]
  ]);
  $checkout_url=reputro_create_stripe_session([
    'amount'=>$price,
    'order_id'=>$order_id
  ]);
  wp_send_json_success(['checkout_url'=>$checkout_url]);
}

Stripe session (псевдо)

function reputro_create_stripe_session($data){
  \Stripe\Stripe::setApiKey(STRIPE_SECRET);
  $session=\Stripe\Checkout\Session::create([
    'mode'=>'payment',
    'line_items'=>[[
      'price_data'=>[
        'currency'=>'usd',
        'unit_amount'=>intval($data['amount']*100),
        'product_data'=>['name'=>'Reputation Campaign Credits']
      ],
      'quantity'=>1
    ]],
    'success_url'=>home_url('/order-success/?oid='.$data['order_id']),
    'cancel_url'=>home_url('/order-failed/?oid='.$data['order_id'])
  ]);
  return $session->url;
}

LocalStorage восстановления

const STORAGE_KEY='reputroOrder';
function saveState(){
  localStorage.setItem(STORAGE_KEY,JSON.stringify({
    q:quantityInput.value,
    speed:deliverySpeedSelect.value,
    url:trustpilotLinkInput.value
  }));
}
function loadState(){
  try{
    const d=JSON.parse(localStorage.getItem(STORAGE_KEY)||'{}');
    if(d.q)quantityInput.value=d.q;
    if(d.speed)deliverySpeedSelect.value=d.speed;
    if(d.url)trustpilotLinkInput.value=d.url;
  }catch(e){}
}
